# mini-projeto-python-Alisson

# Notas do Projeto1
# A função principal desse projeto é a criação de uma lista de produtos
# Dentro desses produtos existe um dicionário com código, nome, preço, quantidade e categoria
# O Sistema começa com um menu com opções numeradas de 0 a 5
# Se o usuário escolher 0 o programa termina
# Escolher '1' permite cadastrar um produto, ditando seu código, nome, preço, quantidade e categoria
# Depois de escrito os dados do produto são cadastrados e salvos na lista
# Existe um sistema de segurança para evitar duplicatas na lista de codigos
# Na opção '2' o sistema lista os dados de todos os produtos salvos
# A opção '3' pede para o usuário informar o código de um produto específico
# Depois de digitar o código, os dados do produto são exibidos
# '4' permite que o usuário altere os dados dos produtos
# Quando o usuário escolhe qual produto ele vai alterar aparece um sub-menu perguntando o que será alterado
# As opções para alteração são '0'-Terminar, '1'-Nome, '2'-Preço e '3'-Quantidade
# Por último, a opção '5' permite o apagamento de um produto da lista
# O sistema pede o código do produto e com isso ele deleta o produto
# Esse sistema pode ser útil para manter uma lista de produtos de uma loja pequena em dia
# Asegurando o monitoramento das vendas

# Notas do Projeto2
# Esse projeto também foca no cadastro de dados em uma lista, nesse caso alunos e suas notas
# Assim como o primeiro projeto, esse projeto começa com um menu interativo
# As opções são numeradas de 0 a 6 e o usuário pode escolher o que fazer até escolher '0' que desliga o programa
# A opção '1' possibilita o cadastro de um aluno
# O usuário precisa colocar o nome e número de matricula para adicionar o aluno na lista 
# A lista de alunos registrados também tem um sistema de segurança para evitar duplicatas
# A opção '2' possibilita inserir notas nos dados dos alunos
# Para inserir as notas é preciso indicar a matrícula do aluno desejado
# Depois de indicar o aluno, é preciso inserir as notas 
# Quando completo as notas são salvas 
# Opção '3' é uma função que lista todos os alunos cadastrados e a média de suas notas
# Opção '4' lista os dados de um aluno específico que o usuário procura
# Opção '5' é uma função importante pois ela mostra os alunos aprovados e reprovados 
# Essa função auxilia um exemplo de uso do programa, ver os status dos alunos  
# A opção '6' cria um sub-menu de relatórios
# Semelhante ao primeiro menu o sub-menu dá opções para o usuário até ele decidir sair dele digitando '0'
# A opção '1' mostra os nomes dos alunos cadastrados 
# Opção '2' mostra médias individuais
# Opção '3' mostra os alunos aprovados e reprovados
# O melhor uso dos sistema é a criação de lista para salvar os dados de estudantes
# Junto disso, o sistema calcula as médias das notas salvas e com isso dita se o aluno está aprovado ou reprovado
